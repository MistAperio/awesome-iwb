# Awesome IWB — 横幅卡片设计文件

本目录存放 Awesome IWB 项目首页横幅卡片的设计源文件，可直接导入 Figma 作为可编辑的矢量设计使用。

## 文件说明

| 文件 | 说明 |
| --- | --- |
| `aiwb-banners.fig.svg` | 包含全部 4 张分类横幅卡片的主设计文件，2x2 网格布局，画布 1328x768。 |

### 卡片内容（从左到右、从上到下）

1. **关于 AIWB** — `#065f46 → #34d399` 绿/青渐变，右侧 “i” 圆形装饰。
2. **屏幕批注与白板软件** — `#1e3a8a → #3b82f6` 蓝色渐变，右侧画笔扇形装饰，右上角项目数徽标 “11 个项目” + 箭头按钮。
3. **课表与看板类软件** — `#0e7490 → #06b6d4` 青色渐变，右侧日历图标，右上角项目数徽标 “11 个项目” + 箭头按钮。
4. **辅助类软件与实用工具** — `#047857 → #10b981` 绿色渐变，右侧齿轮图标，右上角项目数徽标 “33 个项目” + 箭头按钮。

每张卡片尺寸 640x360px，卡片间距 24px，整体置于深色背景 `#0f1117` 之上。

## 在 Figma 中使用

### 方式一：导入 SVG（推荐，可编辑）

1. 在 Figma 中打开目标文件。
2. 顶部菜单选择 **File > Import**（或直接将文件拖入画布）。
3. 选择 `aiwb-banners.fig.svg`。
4. 导入后，每张卡片、每段文字、每个装饰图形都是独立的矢量图层 / 文本图层，可在 Figma 中自由调整颜色、字号、形状与位置。

SVG 中所有文字均使用 `<text>` 元素书写（非转曲），导入后仍为可编辑文本；渐变以 `<linearGradient>` 定义，导入后为可编辑的 Figma 渐变填充。

### 方式二：原生 .fig 二进制格式

本仓库不直接提供 `.fig` 二进制文件。原生 `.fig` 是 Figma 专有的二进制格式，**只能由 Figma 自身导出**：

1. 按上述方式将 SVG 导入 Figma。
2. 在 Figma 中调整满意后，选择 **File > Save local copy…** 即可导出 `.fig` 文件归档。

## 技术备注

- 文字字体：`PingFang SC, Microsoft YaHei, sans-serif`（中文优先）。在未安装这些字体的系统上，Figma 会回退到默认无衬线字体。
- 半透明色统一使用 `fill` + `fill-opacity`（而非 `rgba()`），以保证 SVG 严格合规并兼容 Figma 导入。
- 渐变、画笔、箭头按钮、日历、齿轮等元素均使用基本图元（`rect` / `circle` / `path` / `text`）绘制，便于二次编辑。
